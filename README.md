# Learnsphere -- ASP.NET Core MVC

## Overview

The E-Learning Platform is a complete web-based learning system built
using ASP.NET Core MVC, Entity Framework Core, and SQL Server. It allows
students to subscribe to courses through different package types, attend
private or group sessions, and track their progress.

## Features

### Students

-   Register & login
-   Browse courses
-   Subscribe using multiple package types
-   Attend private & group sessions
-   Track progress
-   Payments with coupons

### Teachers

-   Manage courses and lessons
-   Add package types
-   Create sessions
-   Track student activity

### Admin

-   Manage users
-   Manage courses & categories
-   Manage coupons
-   View analytics

## Architecture (MVC)

-   Controllers
-   Models
-   Views
-   ViewModels
-   Services
-   Repositories

## Tech Stack

-   ASP.NET Core MVC 9
-   EF Core
-   SQL Server
-   Identity
-   Bootstrap
